# 💱 Convertidor de Moneda

Aplicación web para convertir entre diferentes monedas latinoamericanas y mundiales.

## Monedas soportadas
- 🇺🇸 Dólar (USD)
- 🇪🇺 Euro (EUR)
- 🇬🇧 Libra Esterlina (GBP)
- 🇨🇴 Peso Colombiano (COP)
- 🇲🇽 Peso Mexicano (MXN)
- 🇦🇷 Peso Argentino (ARS)
- 🇧🇷 Real Brasileño (BRL)
- 🇨🇱 Peso Chileno (CLP)

## Funcionalidades
- Convertir entre cualquier par de monedas
- Botón para intercambiar monedas rápidamente
- Atajos para conversiones frecuentes
- Muestra la tasa de cambio usada

## Tecnologías
- HTML, CSS, JavaScript

## Nota
Las tasas de cambio están fijas en el código. Para tasas en tiempo real se necesitaría una API externa.
