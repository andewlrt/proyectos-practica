#  Calculadora Web

Una calculadora simple hecha con HTML, CSS y JavaScript.

## ¿Qué hace?
- Suma, resta, multiplicación y división
- Porcentajes
- Borrar último dígito
- Limpiar pantalla

## Tecnologías usadas
- HTML
- CSS (Grid para los botones)
- JavaScript

## Cómo usarla
Solo abre el archivo `index.html` en tu navegador y listo!

## Captura
La calculadora tiene un diseño oscuro con colores morado y rojo.
