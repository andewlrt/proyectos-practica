#  Juego: Adivina el Número

Juego de consola hecho en Python donde tienes que adivinar un número aleatorio.

## Cómo jugar
1. Elige la dificultad (Fácil, Medio, Difícil)
2. El programa genera un número aleatorio
3. Intenta adivinarlo antes de quedarte sin intentos
4. El juego te dice si tu número es muy alto o muy bajo

## Niveles de dificultad
| Dificultad | Rango | Intentos |
|------------|-------|----------|
| Fácil | 1 - 50 | 10 |
| Medio | 1 - 100 | 7 |
| Difícil | 1 - 200 | 5 |

## Cómo ejecutarlo
```bash
python juego.py
```

## Tecnologías
- Python 3
- Módulo `random`
