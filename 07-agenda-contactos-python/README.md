#  Agenda de Contactos

Aplicación de consola en Python para gestionar tus contactos personales.

## Funcionalidades
- Ver todos los contactos
- Agregar contactos nuevos
- Buscar por nombre, apellido o teléfono
- Editar contactos existentes
- Eliminar contactos
- Los contactos se guardan en un archivo JSON

## Cómo ejecutarlo
```bash
python agenda.py
```

## Archivo de datos
Los contactos se guardan automáticamente en `contactos.json` en la misma carpeta.

## Tecnologías
- Python 3
- Módulo `json` (para guardar datos)
- Módulo `os` (para verificar archivos)
