## 🛠️ Tecnologías usadas

- **HTML5** - Estructura de páginas web
- **CSS3** - Estilos, flexbox, grid, gradientes
- **JavaScript** - Lógica, DOM, fetch, localStorage
- **Python 3** - Scripts de consola, manejo de archivos
- **SQL** - Consultas con SQLite
- **MySQL** - Base de datos relacional

---

## 🚀 Cómo usar estos proyectos

### Proyectos web (HTML/JS):
Simplemente abre el archivo `index.html` en tu navegador favorito.

### Proyectos Python:
```bash
python nombre_archivo.py
```
> Requiere Python 3 instalado

### Proyectos SQL:
```bash
# SQLite
sqlite3 base.db < archivo.sql

# MySQL
mysql -u root -p < archivo.sql
```
