# 🧠 Quiz de Tecnología

Quiz interactivo sobre tecnología e informática hecho con HTML, CSS y JavaScript.

## Categorías disponibles
- 💻 Conocimiento General
- 👨‍💻 Programación
- 🌐 Internet y Redes
- 🖥️ Hardware

## Funcionalidades
- 4 categorías con 5 preguntas cada una
- Preguntas en orden aleatorio
- Retroalimentación inmediata (correcto/incorrecto)
- Explicación después de cada respuesta
- Barra de progreso
- Puntaje final con mensaje personalizado

## Tecnologías
- HTML5
- CSS3
- JavaScript

## Cómo usar
Abre `index.html`, elige tu categoría y presiona "Comenzar Quiz".
