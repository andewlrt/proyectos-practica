#  Lista de Tareas

Una aplicación de tareas pendientes hecha con HTML, CSS y JavaScript puro.

## Funcionalidades
- Agregar tareas nuevas
- Marcar tareas como completadas
- Eliminar tareas
- Filtrar por: Todas / Pendientes / Completadas
- Las tareas se guardan en localStorage (no se pierden al recargar)
- Agregar con la tecla Enter

## Tecnologías
- HTML5
- CSS3
- JavaScript (localStorage)

## Cómo usar
Abre `index.html` en el navegador. Las tareas se guardan automáticamente.
